Gec Limitations

The following Eiffel constructs have not been implemented yet:

Storable
does not support classes `MISMATCH_CORRECTOR`, `CLASS_NAME_TRANSLATIONS`, nor the note tag "storable_version" yet
User-defined expanded classes
creation procedure `default_create` is not called
Non-conforming inheritance
feature replication still uses the pre-ECMA semantics of 'select'
SCOOP
using preconditions as wait conditions is not supported yet
Debugger
Profiler
Cecil (C to Eiffel interface)
Eiffel for .Net
Note that this list is not exhaustive.
