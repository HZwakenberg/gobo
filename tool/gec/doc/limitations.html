Gec Limitations

The following Eiffel constructs have not been implemented yet:

Assertions: not monitored at runtime
Garbage Collection: no internal GC, the Boehm GC can be plugged instead
Storable
User-defined expanded classes: semantics of attachement and comparison not fully implemented
Unique constants
Non-conforming inheritance: feature replication still uses the pre-ECMA semantics of 'select'
Multithreading: partially implemented
Debugger
Profiler
Cecil (C to Eiffel interface)
Note that this list is not exhaustive.
